南京地铁每日客流统计
=======================

使用 python 脚本抓取官方微博`@南京地铁` 每天发布的__【昨日客流】__报告，并提取出各线路客流数据。

客流数据起始于 2012-09-07，目前更新至 2015-03-21。[数据地址](https://github.com/waterdrinker/Nanjing-Metro-Passenger-Flow/blob/master/passenger-flow-weibos.txt)，不定期更新。
